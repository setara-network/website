# Setara Website

Official website for [setara.network](https://setara.network).

Built with Next.js + Tailwind CSS.

## Run

```bash
npm install
npm run dev
```

## Pages

- `/` — Home page
- `/whitepaper` — Technical whitepaper
- `/register` — Organization registration
- `/docs/run-node` — Node setup guide
- `/docs/build-app` — Developer documentation
- `/docs/api` — API reference
